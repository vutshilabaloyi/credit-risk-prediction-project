# credit-risk-prediction-project

## Project Overview
This project uses machine learning models to predict if a loan applicant is creditworthy or not. The German Credit data was analyzed, visualized and used to train classification models.

## Objectives
- Explore and understand the dataset.
- Visualize important features.
- Build predictive machine learning models.
- Compare Logistic Regression and Random Forest.
- Evaluate model performance using multiple metrics.

## Dataset
- German Credit Dataset
- 1,000 customer records
- 20 financial and personal features
- Target: Creditability (Good or Bad Credit)

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Machine Learning Models
- Logistic Regression
- Random Forest Classifier

## Model Evaluation
- Accuracy
- Confusion Matrix
- ROC Curve
- AUC Score
- Feature Importance

## Results
- Logistic Regression AUC: **0.79**
- Random Forest AUC: **0.76**

The Logistic Regression model achieved the best performance and generalized better on the German Credit dataset.
